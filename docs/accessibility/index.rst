Barrierefreiheit
================

Cusyist der festen Überzeugung, dass Websites und Software für alle zugänglich
sein sollten, unabhängig von Fähigkeiten oder Beeinträchtigungen. Das Cusy
Design System stellt daher Richtlinien und Best Practices in Bezug auf
Barrierefreiheit bereit.
